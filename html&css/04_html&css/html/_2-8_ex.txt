그래픽 소프트웨어의 종류


드로잉 소프트웨어

선과 도형을 컴퓨터로 그리는 데 적합한 소프트웨어이다. 드로잉 소프트웨어는 일러스트레이션, 심벌마크, 로고, 문자 디자인 등에 많이 사용된다.

일러스트레이터(Illustrator) : 미국 어도비 사가 개발한 그래픽 소프트웨어로, 가장 대표적인 드로잉 소프트웨어이다. 주로 캐릭터와 심벌, 제품 등을 디자인할 때 사용한다.

프리핸드(Free Hand) : 매크로미디어 사에서 만든 드로잉 소프트웨어로, 웹 사이트나 스토리보드를 디자인할 때 사용한다.

코렐드로우(Corel Draw) : 코렐 사에서 만든 그래픽 소프트웨어이다. 드로잉과 이미지 작업을 함께 편집디자인할 수 있는 소프트웨어이다. 코렐드로우는 간단한 문자 디자인이나 복잡한 일러스트레이션 제작에 이르기까지 사용되고 있다.


이미지 소프트웨어

그래픽 자료를 편집디자인하기 위해 이미지를 수정 또는 변형, 이미지와 이미지를 서로 합성할 때 사용되는 소프트웨어이다.

포토샵(Photoshop) : 미국의 어도비 사가 개발한 그래픽 소프트웨어이다. 문자 디자인 또는 입력된 사진이나 일러스트레이션 등의 색상을 보정, 수정 또는 합성, 복원하는 데 사용한다.

페인터(Painter) : 일러스트레이션을 전문적으로 제작하는 사람들이 주로 사용하는 소프트웨어이다. 연필, 지우개, 붓, 펜파스텔, 목탄, 수채화 등 다양한 도구들이 지원되어 표현 기법을 다르게 하는 일러스트레이션 제작을 위해 널리 사용되고 있다.

페인트샵 프로(Paint Shop Pro, PSP) : 마이크로소프트 사에서 개발한 그래픽 편집 소프트웨어로, 그래픽을 편집디자인하거나 화면 캡처 또는 애니메이션 작업을 할 때 사용한다. 편집 소프트웨어에 내장된 일반적인 기능들이 있어서 포토샵과 함께 많이 사용되고 있다.


편집 소프트웨어

신문, 잡지, 단행본과 같은 일반적인 출판물을 편집 또는 디자인하는 데 사용되는 소프트웨어이다.

쿽익스프레스(QuarkXPress) : 미국의 쿽(Quark) 사에서 개발한 편집디자인 소프트웨어이다. 문자, 레이아웃, 컬러, 일러스트레이션 등을 편리하게 사용할 수 있으며 대표적인 출판 편집용 소프트웨어이다.

페이지메이커(PageMaker) : 미국 앨더스 사에서 처음 개발한 페이지 레이아웃 소프트웨어로, 개인용 탁상 출판 소프트웨어이다.

인디자인(InDesign) : 미국 어도비 사에서 개발한 출판 레이아웃 소프트웨어로, PC와 매킨토시에서 모두 사용할 수 있다. 비슷한 소프트웨어로는 쿽익스프레스, 페이지메이커 등이 있다.
